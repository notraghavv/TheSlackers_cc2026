# MP REVIVE

// A futuristic interactive tourism experience based on Madhya Pradesh, built with p5.js. //

##  About The Project

**MP REVIVE** is a cyberpunk-style interactive experience set in the year **2150**, where Madhya Pradesh has fallen into environmental and cultural collapse.

Players must revive the state by restoring four major regions through an interactive hacking system. Once revived, each region unlocks historical, cultural, and tourism information about famous cities and landmarks.

The project combines:
- Tourism awareness
- Storytelling
- Interactive exploration
- Futuristic UI/UX
- Environmental restoration themes

---

##  Features

- Interactive map of Madhya Pradesh
- Futuristic cyberpunk interface
- Hold-to-hack gameplay mechanic
- Dynamic animations and visual effects
- Region exploration system
- Landmark slideshow cards
- Background music system
- Keyboard-based navigation
- Story-driven intro sequence
